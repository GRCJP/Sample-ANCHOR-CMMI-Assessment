## Core Skill Set

### 1. Program Oversight
- Monitor statewide maturity across all 90 agencies from the State Cybersecurity Dashboard
- Track completion percentage, in-progress count, and unresponsive agencies
- Review historical trend lines (year-over-year maturity growth)
- Identify at-risk agencies via the Top Performing vs. At-Risk chart

### 2. Agency Portfolio Management
- Use Agency Assessment Management to view, filter, and manage the full agency list
- Filter by Status, Exec Branch, DoIT Enterprise, Size Tier, IT Complexity, Maturity Group, Wave
- Create new system assessments via "+ New Assessment"
- Send batch intake reminders to pending agencies
- Export master tracker to Excel

### 3. Scheduling & Pipeline
- Navigate the 2-Year Pipeline to review the 90-agency 24-month schedule
- Review wave assignments, pod capacity, and throughput KPIs
- Monitor SLA compliance across rolling 90-day window
- Sync pipeline schedule to ServiceNow

### 4. Workflow Configuration
- Select and assign assessment templates (Standard Enterprise, Small Agency, OT/ICS, etc.)
- Configure lifecycle rules and workflow stages per agency type
- Set automated escalation rules (overdue evidence, unresponsive agencies)

### 5. Compliance Metrics Review
- Review all statewide compliance metrics in the dashboard
- Review CMMI scores by CSF function vs. DoIT target baseline
- Export compliance data to ServiceNow GRC (JSON/CSV/XML)

### 6. Settings & Administration
- Manage IAM users — invite, deactivate, change roles
- Configure ServiceNow GRC integration (URL, OAuth, sync frequency)
- Review platform activity log and audit trail
- Configure email/notification settings
